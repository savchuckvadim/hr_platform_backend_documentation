# Видеозвонки (ОПЦИОНАЛЬНО)

## Описание

Опциональный модуль для проведения видеозвонков прямо в системе.
Возможность созваниваться для собеседований без использования сторонних сервисов.
Интеграция с WebRTC или специализированными сервисами для видеозвонков.

## Ключевые концепции

- Создание видеозвонков между кандидатом и работодателем
- WebRTC для peer-to-peer соединений
- Или использование специализированных сервисов (LiveKit, Twilio)
- Планирование звонков через календарь
- Управление комнатами для групповых звонков
- Интеграция с модулем уведомлений
- Запись звонков для последующего просмотра (опционально)

> **Примечание**: Этот модуль является опциональным и может быть реализован в будущем
