# Core Events Module

## Описание

Централизованный модуль событий для всей системы. Обеспечивает типизированную публикацию и подписку на события между модулями. Любой модуль может импортировать enum событий, знать тип payload, подписаться на события - всё централизованно.

## Ключевые концепции

- Централизованный источник правды для типов событий
- Типизированная публикация и подписка (автокомплит, проверка типов)
- EventBus как абстракция (может работать локально, через BullMQ, Redis, Kafka)
- Автоматическая подписка через декораторы
- Изоляция бизнес-логики от инфраструктуры (модули не знают про Bull)
