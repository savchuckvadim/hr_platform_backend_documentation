# Структура модуля - луковая архитектура

## Описание

Описание организации модулей проекта по принципам луковой (onion) архитектуры.
Разделение на слои: domain, application, infrastructure.
Принципы изоляции и зависимостей между слоями.

## Ключевые концепции

- Domain слой - бизнес-логика и сущности
- Application слой - use cases и сервисы
- Infrastructure слой - репозитории, внешние сервисы
- Dependency Injection для связывания слоев
- Изоляция бизнес-логики от инфраструктуры
